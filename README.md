# Quantum Trade AI

A sophisticated AI-powered trading platform built with React and Vite.

## Features

- Advanced technical analysis with interactive charts
- AI-driven trading decisions and market analysis
- Real-time portfolio management and risk assessment
- Comprehensive options and crypto trading capabilities
- Social trading and influencer monitoring
- Advanced backtesting and simulation tools

## Running the app

```bash
npm install
npm run dev
```

## Building the app

```bash
npm run build
```

## Tech Stack

- React 18 with Vite
- Tailwind CSS for styling
- Recharts for data visualization
- Framer Motion for animations
- Radix UI components
- React Router for navigation
